# 英文稿件专业润色技能

## 技能来源
本技能完全基于 **Hilary Glasman-Deal** 所著 *"Science Research Writing for Non-Native Speakers of English"* (Imperial College Press, 2010) 的写作原则、语法规范和润色方法。每一项修改规则和判断标准均可追溯到书中的具体章节。

---

## 一、技能概述

本技能对英文科研/学术/技术文本进行全面多层次润色，严格遵循Glasman-Deal书中的写作模型、语法规则和词汇使用标准。输出为带有详细修改标注的Word文档，使用颜色编码区分不同类型的修改，并附带修改说明报告。

---

## 二、润色前分析

### 2.1 文本类型识别
首先判断文本属于论文的哪个部分（基于书的IMRaD模型）：

| 部分 | 模型来源 | 关键特征 |
|------|----------|----------|
| Introduction | Unit 1 | 从宽到窄，4组件 |
| Methodology | Unit 2 | 被动语态为主，菜单式 |
| Results | Unit 3 | 主观评价+数据，菜单式 |
| Discussion/Conclusion | Unit 4 | 从窄到宽，4组件 |
| Abstract | Unit 5 | Model 1或Model 2 |
| Title | Section 5.6 | 简洁，句子式大小写 |

### 2.2 确定文体和读者
- 国际期刊论文 → 所有模型和规则均适用
- 学位论文 → 被动语态为主(Section 1.2.3)
- 研究报告 → 可灵活但遵循核心原则
- 一般学术写作 → 应用通用的清晰度、连贯性规则

---

## 三、多层次润色流程

按照书中的优先级顺序执行，先解决影响理解的问题，再优化表达质量。

---

### 层级一：结构层面检查

#### 3.1 整体结构验证

**Introduction结构（Unit 1, Section 1.3.3）：**
```
□ 是否以建立研究重要性开始？
□ 是否提供背景事实信息？
□ 是否有文献综述部分？
□ 是否明确定位研究空白？
□ 是否描述本论文/本研究？
```

**Methodology结构（Unit 2, Section 2.3.3）：**
```
□ 是否有概述/引入？
□ 是否提供精确细节（数量、温度、时间、条件等）？
□ 是否为选择提供理由（justification）？
□ 是否展示谨慎操作？
□ 是否与已有研究关联？
□ 是否指出问题/局限性（如适用）？
```

**Results结构（Unit 3, Section 3.3.3）：**
```
□ 是否回顾研究目的或已有研究？
□ 是否有结果总览？
□ 是否邀请读者查看图表？
□ 是否对关键结果进行主观评价？
□ 是否与他人结果比较？
□ 是否提及结果中的问题？
□ 是否向含义/讨论过渡？
```

**Discussion/Conclusion结构（Unit 4, Section 4.3.3）：**
```
□ 是否回顾前文（Introduction/Methodology/Results）？
□ 是否总结关键结果？
□ 是否将本研究映射到已有研究图景？
□ 是否明确陈述成就/贡献？
□ 是否深化含义和可能应用？
□ 是否讨论局限性？
□ 是否提出未来研究方向？
```

**Abstract结构（Unit 5, Section 5.3.3）：**
```
□ Model 1: Background + Aim + Method + Results + Implications
□ Model 2: 论文做什么 + 方法 + 关键结果
□ 是否为独立可理解的文本？
```

#### 3.2 段落组织（Section 1.2.4 Paragraphing）
- [ ] 每段以主题句（topic sentence）开头
- [ ] 一段围绕一个主题展开
- [ ] 无单句段落
- [ ] 无超长段落
- [ ] 主题转换时开始新段落

---

### 层级二：句子层面检查

#### 3.3 句子连接（Section 1.2.2 Signalling Language）

检查每对相邻句子之间的连接方式：

**四种连接方式：**
1. **重叠**: 重复前句的关键词
2. **代词/代用词**: it, they, this method, these systems
3. **分号或关系从句**: 紧密相关的句子
4. **信号连接词**: 明确关系

**信号连接词检查：**
- 因果关系：due to, because, since, on account of, in view of
- 结果关系：therefore, consequently, hence, as a result, which is why
- 对比关系：however, whereas, while, but, by contrast, on the other hand
- 意外关系：although, even though, despite, nevertheless, nonetheless, yet
- 添加关系：in addition, moreover, furthermore, also, apart from that

**书中特别提醒：**
- ⚠️ `since`也可表示"从那时起"，有歧义时避免使用
- ⚠️ `while`也可表示"同时"，有歧义时避免使用
- ⚠️ `so`太口语化，不要在正式写作中用于句首表结果
- ⚠️ `on the contrary`和`conversely`表示"恰恰相反"，而非仅是"不同"
- ⚠️ `besides`比列表中的其他添加词更有说服力

#### 3.4 句子结构优化

**检查并修正：**
- [ ] 超长句子（>40词）→ 考虑拆分
- [ ] 歧义结构（特别是副词位置，Section 2.2.3）
- [ ] 主语和动词距离过远
- [ ] 多个which从句嵌套
- [ ] 复合名词过多造成理解困难（Section 5.6）

**副词位置规则（Section 2.2.3）：**
- 修饰整句的副词放在句首
- 避免副词簇造成歧义
- 必要时将长句拆分为短句，每个带自己的副词

### 层级三：语法精确性检查

#### 3.5 时态使用（书中核心语法）

**Introduction时态（Section 1.2.1）：**
- 背景事实 → Present Simple
- 文献综述中具体研究 → Past Simple (showed, demonstrated, found)
- 研究空白（与现在相关）→ Present Perfect (little attention has been paid)
- 本论文描述 → Present Simple (This paper presents)

**Present Simple vs Present Continuous（Section 1.2.1）：**
- Present Simple = 永久情况/公认事实
- Present Continuous = 暂时情况
- 科研中几乎总是用Present Simple表示事实

**Past Simple vs Present Perfect区分（Section 1.2.1）：**
- Past Simple: 与现在无关的历史事件（I lived in Tokyo... but I don't anymore）
- Present Perfect: 与现在仍然相关的事件（I have lived in Tokyo... and I still live there NOW）
- **关键判断**：是否与"现在"有关？有关→Present Perfect；无关→Past Simple

**Results时态（Section 3.4.2）：**
- Past Simple: We found that x occurred（保守）
- Present Simple: We find that x occurs（强声明，赋予真理地位）
- 选择影响读者对结果可靠性的判断

**Methodology时态（Section 2.2.1）：**
- 标准程序 → Present Simple被动 (is collected, is inserted)
- 你实际做的 → Past Simple被动 (was collected, was inserted)
- ⚠️ **这是书中最强调的"隐藏错误"** — 语法正确但时态选择不当
- 必须用in this study, in our experiments等标记自己的贡献

**Abstract时态（Section 5.2.1）：**
- 论文本身做什么 → Present Simple
- 你做了什么 → Past Simple（或Present Simple用于方程/计算）
- 结果 → Past Simple或Present Simple取决于确信程度
- 成就 → Present Perfect (We have demonstrated)
- 应用 → Present Simple

#### 3.6 冠词检查（Section 2.2.2 a/an/the）

**核心判断标准：**
> USE `THE` IF OR WHEN YOU AND YOUR READER BOTH KNOW WHICH THING/PERSON YOU MEAN.

**应用规则：**
- [ ] 第一次提及→ a/an；再次提及→ the
- [ ] 唯一指代→ the (the sun, the environment, the capital)
- [ ] 不重要/不知道/读者不知道→ a/an
- [ ] 共享知识→ the，即使首次提及(the check-in after Heathrow Airport)
- [ ] 通用真理可用the, a/an, 或∅+复数

**a vs an：**
- a + 辅音音素（a UV light /ju:/）
- an + 元音音素（an MRI scan /em/）

**特殊含义变化（Section 2.2.2）：**
```
"This effect may hide a connection" = 可能存在某种联系
"This effect may hide the connection" = 存在一个确定的联系
```

#### 3.7 情态动词检查（Section 4.2）

| 情态动词 | 含义 | 用法 |
|----------|------|------|
| may/might/could | 可能 | might稍弱于may |
| should/ought to | 很可能 | 基于推理 |
| must | 几乎确定 | 但缺乏实证 |
| cannot | 不可能 | 不是"可能不" |
| can | 能够 | 表示能力 |
| need to/have to | 必要 | must not = 禁止 ≠ 不必要 |

**关键区分（Section 4.2）：**
- `may not` = 可能不 (possibility negative)
- `cannot` = 不可能 (impossible)，完全不同！

#### 3.8 被动/主动语态（Section 1.2.3 & 2.2.1）
- [ ] 研究团队→可用we
- [ ] 个人研究→用被动或dummy subject (This paper/This study)
- [ ] 自己的操作与标准程序能清楚区分
- [ ] agentless passive不造成歧义时可用

### 层级四：词汇精准度提升

#### 3.9 模糊词汇替换

| 原词 | 书中推荐替换 |
|------|-------------|
| did/showed/found | demonstrated, established, revealed, identified (Unit 1词汇表) |
| very big/large | substantial, considerable, significant |
| very small | negligible, marginal, slight |
| more and more | increasing, growing |
| important | crucial, essential, vital, fundamental |
| different | distinct, contrasting, dissimilar |
| similar | analogous, comparable, in line with |
| because of | due to, on account of, in view of |

#### 3.10 数量/频率语言精确化（Section 3.2.2 & 3.2.3）

**频率表达（从高到低）：**
```
100%: always, invariably, without exception, on every occasion
 90%: generally, normally, usually, as a rule
 70%: frequently, often, commonly
 50%: as often as not (中性)
 30%: sometimes, on some occasions
 10%: occasionally, now and then, from time to time
  5%: rarely, seldom, infrequently
  1%: hardly ever, almost never, scarcely ever
  0%: never, on no occasion, at no time
```

**数量语言对读者感知的影响：**
- 增大数量：most, a great deal, considerable, substantial, as many as 45
- 减小数量：only, barely, a few, negligible, as few as 45
- 强调：extremely, by far, particularly, especially, even higher/lower
- 接近：approximately, nearly, close to, virtually, practically
- 模糊/中立：some, fairly, quite, reasonably, relatively, somewhat

#### 3.11 因果关系表达（Section 3.2.4）
- 强因果关系：caused, produced, is due to, resulted in
- 部分原因：is a factor in, contributed to
- 初始原因：originated in, initiated
- 弱因果关系：is linked to, is related to, is associated with（加*表示可能仅是关联）
- 弱化论断：It appears that, It seems that, may have been, could be

#### 3.12 各部分词汇应用

**Introduction词汇检查（Section 1.4）：**
- [ ] 重要性建立使用books词汇
- [ ] 文献综述动词精确
- [ ] 研究空白词汇恰当且礼貌
- [ ] 本论文描述清晰

**Methodology词汇检查（Section 2.4）：**
- [ ] 使用精确动词而非通用的was done/used
- [ ] 空间位置词准确
- [ ] 理由说明词适当
- [ ] 谨慎操作词充分
- [ ] 问题表达专业（见Section 2.4.2第7部分）

**Results词汇检查（Section 3.4）：**
- [ ] 客观描述+主观评价结合
- [ ] 比较词汇精确（in line with vs identical to vs consistent with）
- [ ] 问题最小化表达（minor deficit, slightly, negligible）

**Discussion/Conclusion词汇检查（Section 4.4）：**
- [ ] 映射词汇准确
- [ ] 成就有力的!替代词（compelling, striking, remarkable, unprecedented）
- [ ] 局限性→未来工作词汇搭配恰当
- [ ] future work should（邀请他人）vs future work will（自己计划）区分

- [ ] 无感叹号（用词汇power传递兴奋感，Section 3.3.2 Key）

## 四、输出Word文档规范

### 4.1 文档命名

### 4.2 修订颜色编码

| 颜色 | 含义 | 示例 |
|------|------|------|
| 🔴 红色 | 删除/修正的内容 | 语法错误、拼写错误、标点修正 |
| 🟢 绿色 | 添加的内容 | 补充的连接词、缺失的冠词 |
| 🔵 蓝色 | 替换的内容 | 词汇升级、句子结构优化 |
| 🟣 紫色 | 格式/风格调整 | 段落重组、正式度调整 |

## 五、全书重点错误排查清单

### Critical Issues（可能导致论文被拒）

- [ ] **Methodology时态混淆**：标准程序和自己操作的时态未区分（Section 2.2.1）—这是"隐藏错误"，语法正确但含义错误
- [ ] **Introduction研究空白时态**：未使用Present Perfect而用了Past Simple（Section 1.2.1）—读者可能认为问题已解决
- [ ] **Results无主观评价**：仅描述数据未给予解释（Section 3.2.3）—"Results do not speak for themselves"
- [ ] **Discussion未陈述成就**：害羞或遗漏（Section 4.3.2）
- [ ] **无文献回顾**：Introduction中缺少对前人研究的梳理

### Major Issues（显著影响质量）

- [ ] **句子间缺乏连接**：每句之间的"危险空间"未封闭（Section 1.2.2）
- [ ] **冠词使用不当**：a/the/∅混淆
- [ ] **未提供选择理由**：Methodology中只说做了什么未说为什么
- [ ] **参考文献位置不当**：credit归属错误
- [ ] **段落无主题句**：读者无法快速获取信息
- [ ] **Abstract不能独立理解**：对未读全文的读者无意义

### Minor Issues（改进空间）

- [ ] 信号语单调（仅用however和therefore）
- [ ] 数量/频率词不够精确
- [ ] 成就词不够有力
- [ ] 局限性未与未来工作有效关联
- [ ] 标题复合名词过长或歧义

## 六、书中"幽默对照表"提醒（Section 3.4.2）

本书提供了一个科研写作的"潜台词对照表"，润色时应确保文字表达的是真实含义而非相反：

| 你写的 | 可能被解读为 |
|--------|------------|
| It has long been known that... | I can't remember the reference |
| Typical results are shown | Only the best results are shown |
| Three of the samples were chosen for detailed study | The results of the others didn't make sense |
| It is suggested that... / It is believed that... | I think that... |
| Agreement was excellent/good/satisfactory | Actually fair/poor/awful |
| Unfortunately, a quantitative theory has not been formulated | Neither does anyone else |
| It is clear that much additional work is required | I don't understand it |

## 七、质量保证

1. 每一项修改必须在Glasman-Deal书中找到明确依据
2. 不改变作者的学术含义和结论
3. 不确定的修改标注"建议考虑"而非直接修改
4. 保持作者的声音和风格（在符合书中规则的前提下）
5. 对不同类型的文本应用不同的润色标准
